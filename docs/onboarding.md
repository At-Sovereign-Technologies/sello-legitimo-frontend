# Guía de incorporación

## Objetivo
Este repositorio contiene la interfaz web de Sello Legítimo (Vite + React + TypeScript).

## Inicio rápido
1. Abre el repositorio en VS Code.
2. Opcional: Reopen in Container para entorno aislado.
3. Instala dependencias:

```bash
npm install
```

4. Inicia el servidor de desarrollo:

```bash
npm run dev -- --host 0.0.0.0 --port 5173
```

5. Abre la aplicación en el puerto 5173.
