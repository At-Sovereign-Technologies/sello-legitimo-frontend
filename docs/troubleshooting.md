# Solución de problemas

## El puerto 5173 no está disponible
Inicia Vite en otro puerto o cierra el proceso que lo ocupa.

## Error al autenticar
Actualmente el frontend espera endpoints en /api/v1/auth. Si el backend expone otro contrato, la integración E2E puede fallar.

## Lint
El comando npm run lint debe ejecutarse sin errores tras esta normalización de incorporación.
